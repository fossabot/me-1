# Dependencies

```
brew install nvm
nvm install 8
npm install -g gulp-cli
```

# Installation

```
npm install
```

# Usage

```
npm start
```

# Testing

```
npm test
```

# Generation

```
npm run build
npm run resume
```

# Publishing

```
npx resume register
npx resume login
npx resume publish
```
