The backing markup and styles for [`jsonresume-theme-randytarampi`](../jsonresume-theme) and [`www`](../www).

# Dependencies

```
brew install nvm
nvm install 8
npm install -g gulp-cli
```

# Installation

```
npm install
```

# Usage

```
npm start
```

# Testing

```
npm test
```

# Generation

```
npm run build
npm run resume
```
